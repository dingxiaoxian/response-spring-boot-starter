# Response Spring Boot Starter

## usage

### maven

```xml

<dependency>
    <groupId>tech.xiaoxian</groupId>
    <artifactId>response-spring-boot-starter</artifactId>
    <version>1.0.0</version>
    <type>pom</type>
</dependency>
```

### gradle

```groovy
implementation 'tech.xiaoxian:response-spring-boot-starter:1.0.0'
```
